#  Insurance Data Analysis (EDA Project)

## Overview

This project is a simple yet insightful **Exploratory Data Analysis (EDA)** on an insurance dataset. The main goal was to understand how different factors like age, BMI, smoking habits, and region influence medical insurance charges.

Instead of just jumping into machine learning, this project focuses on **understanding the data first**, which is a crucial step in any data science workflow.

---

##  About the Dataset

The dataset contains basic information about individuals such as:

* Age
* Gender
* BMI
* Number of children
* Smoking status
* Region
* Insurance charges

These features help in analyzing what really drives higher insurance costs.

---

## What I Did in This Project

###  Data Cleaning

* Checked for missing values and duplicates
* Cleaned and structured the dataset for analysis

### Data Transformation

* Converted categorical data (like gender and smoker) into numerical form
* Applied encoding techniques for the region column

### Data Visualization

* Used graphs and plots to explore the data
* Identified patterns, trends, and possible outliers
* Created a correlation heatmap to understand relationships between variables

###  Preprocessing

* Scaled numerical features to bring them to a common range
* Prepared the dataset for future machine learning models

---

##  Key Observations

* Smoking has a **major impact** on insurance charges
* Age and BMI also play an important role
* Charges are not evenly distributed (skewed data)
* The dataset is clean and suitable for predictive modeling
